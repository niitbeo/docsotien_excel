⚠️ NEU GAP LOI
═══════════════════════════════════════════════════════

LOI: "Trust access to VBA project"
GIAI PHAP:
  1. Mo Excel
  2. File → Options → Trust Center → Trust Center Settings
  3. Macro Settings
  4. Tich vao: "Trust access to the VBA project object model"
  5. Nhan OK
  6. Chay lai DocTien_AutoInstaller.exe

GIAI PHAP:
