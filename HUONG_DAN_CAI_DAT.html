📊 DocTien cho Excel

Chuyển số thành chữ tiếng Việt tự động

1

Mở VBA Editor

Trong Microsoft Excel, nhấn phím tắt:

Alt + F11

Cửa sổ "Microsoft Visual Basic for Applications" sẽ hiện ra.

2

Import File DocTien.bas

Trong VBA Editor:

  • Chọn menu File → Import File...
  • Tìm đến file DocTien.bas
  • Đường dẫn: C:\Users\nltruong\Desktop\doctien\DocTien.bas
  • Nhấn Open
3

Lưu File Excel

File Excel của bạn phải được lưu dạng .xlsm (Macro-Enabled):

  • Nhấn Ctrl + S hoặc File → Save As
  • Chọn định dạng: Excel Macro-Enabled Workbook (*.xlsm)
  • Nhấn Save
✅ Xong! Giờ bạn có thể dùng hàm =DocTien() trong file Excel này!
🎨 Ví dụ sử dụng

Trong Excel, tại ô B2, gõ công thức:

=DocTien(A2)
Cột A (Số) Cột B (Công thức) Kết quả
12345 =DocTien(A2) Mười hai nghìn ba trăm bốn mươi lăm đồng
1000000 =DocTien(A3) Một triệu đồng
2500.50 =DocTien(A4) Hai nghìn năm trăm đồng phẩy năm mươi đồng
⚠️ Nếu gặp lỗi #NAME?

Lỗi này có nghĩa là Excel chưa nhận diện được hàm DocTien().

Giải pháp:

  • Kiểm tra lại Bước 2 - đã import file DocTien.bas chưa?
  • File Excel đã lưu dạng .xlsm chưa?
  • Thử đóng Excel và mở lại
  • Nếu có cảnh báo "Security Warning", nhấn Enable Content
💡

Dùng cho MỌI file Excel (Optional)

Để hàm DocTien() có sẵn trong mọi file Excel, cài vào Personal Macro Workbook:

  1. Mở Excel → nhấn Alt + F11
  2. Trong VBA Editor, tìm "VBAProject (PERSONAL.XLSB)" bên trái
  3. Nếu không thấy, tạo mới:
    • View → Macros (Alt + F8)
    • Gõ tên bất kỳ → chọn "Personal Macro Workbook" → Create
  4. Import file DocTien.bas vào PERSONAL.XLSB
  5. Lưu lại (Ctrl + S)

Lợi ích: Từ giờ mở Excel nào cũng có sẵn hàm DocTien()!
